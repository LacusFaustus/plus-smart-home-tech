# Smart Home Tech - Микросервисная архитектура

## Описание проекта

Проект представляет собой интернет-магазин для технологий умного дома, построенный на микросервисной архитектуре.

### Сервисы

#### Инфраструктура
- **discovery-server** (Eureka) - порт 8761
- **config-server** - порт 8888
- **gateway** - порт 8080 (единая точка входа)

#### Бизнес-сервисы
- **shopping-store** - управление товарами
- **shopping-cart** - управление корзиной
- **warehouse** - управление складом
- **order** - управление заказами
- **payment** - обработка платежей
- **delivery** - управление доставкой

### Полный цикл заказа

1. Пользователь добавляет товары в корзину (shopping-cart)
2. Создается заказ (order)
3. Товары резервируются на складе (warehouse)
4. Создается платеж (payment)
5. После успешной оплаты заказ собирается (warehouse)
6. Создается доставка (delivery)
7. Заказ доставляется
8. Заказ завершается

## Запуск проекта

### Предварительные требования

- Docker и Docker Compose
- Java 21
- Maven 3.8+

### Быстрый запуск (рекомендуемый)

```bash
# Собрать все сервисы
mvn clean package -DskipTests

# Запустить все сервисы через Docker Compose
docker-compose up -d
